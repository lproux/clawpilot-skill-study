# Success Rubric: Customer Renewal Email -> Forecast Impact + Path

**Workflow:** WF-35 | **Mode:** Co-Work | **Actor:** CSA/DSSP

## Expected Output

Contract state summary with freshness, forecast impact with confidence, response draft (no commercial commitments), path forward with MACC/funding options if applicable

## Routing Requirements

- **Entry skill:** customer-meeting-prep-outreach-coach
- **Modules:** email-chat-context
- **Conditional handoffs:**
  - funding-partner-activation-coach (when: contract/billing lookup needed)
  - forecast-business-review-coach (when: forecast impact assessment needed)

## Quality Gates

- Output must be actionable (named owners, specific next steps)
- Evidence must be cited with freshness labels
- No fabricated data points
- Routing must match contract unless scenario is design-redesign intent
- Guardrails respected (draft-only, no unauthorized writeback)
