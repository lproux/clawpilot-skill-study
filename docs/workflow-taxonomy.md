## Modes

- **Automation:** System-triggered, runs on schedule or event without explicit user invocation
- **Co-Work:** User and system collaborate interactively on a multi-step task
- **Prompt:** User asks a question or requests a deliverable; system produces output

## Tiers

- **Tier 1 (Simple):** 5-7 failure modes, straightforward routing, single-skill resolution
- **Tier 2 (Standard):** 7-10 failure modes, may involve handoffs, moderate complexity
- **Tier 3 (Complex):** 10-14 failure modes, multi-skill routing, rich output shapes
- **Tier 4 (Max complex):** 14-20 failure modes, trigger-organized, cross-skill synthesis required
